# SOAP-JAXWS
A simple SOAP Service and Client, based on JAX-WS.
Developed in Eclipse, using Java 1.8 (8) and WildFly 10.x

Used for Server:
	Step-by-step SOAP web services example in Java using Eclipse

Used for Client:
